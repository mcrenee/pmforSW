# 肆瓦项目管理器 v1.0

## 🎯 项目简介

**肆瓦项目管理器**是上海肆瓦商业管理有限公司专用的RBO项目流程管理与数据登记系统。系统采用**上下分离布局**，首页直接呈现完整的工作流程图和数据登记管理系统，帮助团队高效管理浦东DRC、西安RBO、南京RBO三个项目的报数打款流程。

## ✨ 核心特色

### 📐 上下分离布局设计
- **上半部分**：工作流程图（从左到右展示执行流程）
- **下半部分**：数据登记管理系统（14个字段记录管理）
- **一站式访问**：首页直接呈现两大核心功能，无需额外导航

- 🏢 **浦东DRC项目**

- 🔄 **项目切换**：通过标签页切换不同项目流程

- 📄 **参考文件**：标注OA提报的参考文件名称

## 📖 使用指南

### 1. 查看工作流程图
1. **打开首页**：访问系统首页
2. **上半部分**：查看工作流程图
3. **切换项目**：点击标签页切换浦东DRC或西安/南京RBO
4. **查看详情**：鼠标悬停在流程卡片上查看完整邮件模板和平台地址

### 2. 登记处理信息
1. **下半部分**：数据登记管理区域
2. **新增记录**：点击"新增记录"按钮，填写14个字段
3. **搜索筛选**：使用搜索框和筛选器快速查找
4. **编辑/删除**：点击表格中的操作按钮

### 3. Excel导入导出
- **导出**：点击"导出Excel"按钮，下载所有记录
- **导入**：点击"导入Excel"按钮，选择Excel文件批量导入

## 💡 典型使用场景

### 场景1：每月1号报数（浦东DRC）
```
1. 查看流程图（上半部分）
   → 点击"浦东DRC项目"标签
   → 查看肆瓦的邮件模板和MC的平台地址

2. 执行任务
   → 肆瓦：按模板发送报数邮件
   → MC：登录平台上传Excel文件（注意12点规则）

3. 登记信息（下半部分）
   → 点击"新增记录"
   → 填写处理日期、RBO编码、分成周期等
   → 保存记录
```

### 场景2：每月2号打款（西安/南京RBO）
```
1. 查看流程图（上半部分）
   → 点击"西安/南京RBO项目"标签
   → 查看第二阶段流程

2. 执行任务
   → 肆瓦：按打款凭证邮件模板发送（含附件）
   → MC：在OA提报打款邮件和凭证文件

3. 更新记录（下半部分）
   → 搜索对应的RBO记录
   → 点击"编辑"
   → 更新资金流OA单和资金流处理日期
   → 将状态改为"已完成"
```

### 场景3：月度数据统计
```
1. 使用筛选功能（下半部分）
   → 选择日期范围
   → 选择状态（如：已完成）

2. 查看统计面板
   → 总记录数
   → 已完成数
   → 总打款金额

3. 导出报告
   → 点击"导出Excel"
   → 在Excel中进行详细分析
```

## 📊 数据说明

### 数据存储
- 所有数据保存在浏览器的 **LocalStorage** 中
- 数据会自动保存，无需手动保存
- 即使关闭浏览器，数据也不会丢失
- 建议定期导出Excel文件作为备份

### 数据备份建议
- **每周导出**：定期导出Excel文件保存
- **重要操作前**：在批量删除或修改前先导出备份
- **云端备份**：将导出的Excel文件保存到云盘
