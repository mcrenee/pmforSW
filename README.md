# RBO管理系统 v3.0

## 🎯 项目简介

**RBO管理系统**是一个专业的固定期限RBO处理流程与登记管理平台，专为浦东DRC、西安RBO、南京RBO三个项目设计。系统包含两个核心模块：**RBO登记管理器**和**RBO流程图系统**，帮助团队高效管理报数打款流程。

## ✨ 核心模块

### 1️⃣ 📊 RBO登记管理器
**完整的报数打款登记与追踪系统**

**核心功能：**
- ✍️ **记录管理**：新增、编辑、删除RBO处理记录
- 📋 **完整字段**：14个关键字段全覆盖
  - 处理日期、状态、系统
  - RBO编码、内部RBO编码、简称
  - 分成周期开始/结束日
  - 信息流OA、信息流处理日期
  - 应打款金额、实际打款金额
  - 资金流OA单、资金流处理日期
- 🔍 **智能搜索**：按RBO编码、简称、内部编码快速搜索
- 🎯 **多维筛选**：状态筛选、日期筛选
- 📊 **实时统计**：总记录数、已完成数、处理中数、总打款金额
- 📥 **Excel导出**：一键导出所有记录为Excel文件
- 📤 **Excel导入**：批量导入Excel数据
- 💾 **本地存储**：数据自动保存到浏览器本地存储

**数据字段说明：**

| 字段 | 说明 | 示例 |
|------|------|------|
| 处理日期 | 流程处理的日期 | 2025-08-06 |
| 状态 | 已完成/处理中/待处理 | 已完成 |
| 系统 | 所属系统 | 二合一 |
| RBO编码 | 外部RBO编码 | RT-CN0100-PAWSV0001 |
| 内部RBO编码 | 内部编码 | R0000000003 |
| 简称 | 项目简称 | 上海肆瓦南京机场（二合一） |
| 分成周期开始日 | 分成周期起始 | 2025-07-30 |
| 分成周期结束日 | 分成周期结束 | 2025-07-31 |
| 信息流OA | 信息流OA单号 | IT06-20250801001 |
| 信息流处理日期 | 信息流处理日期 | 2025-08-06 |
| 应打款金额 | 应支付金额 | ¥0.00 |
| 实际打款金额 | 实际支付金额 | ¥0.00 |
| 资金流OA单 | 资金流OA单号 | / |
| 资金流处理日期 | 资金流处理日期 | 2025-08-06 |

**功能亮点：**
- 🎨 现代化表格设计，清晰易读
- 📱 完全响应式，移动端友好
- ⚡ 实时数据更新，无需刷新
- 🔄 与流程图系统无缝衔接
- 💡 表单验证，防止数据错误

### 2️⃣ 🗺️ RBO流程图系统
**交互式流程图展示固定期限RBO处理流程**

**支持项目：**
- 浦东DRC项目
- 西安RBO项目
- 南京RBO项目

**核心功能：**
- 🔄 **项目模块切换**：支持不同项目流程的独立查看
- 🎯 **时间节点标注**：清晰展示每月1号/15号、2号/16号等关键时间点
- 👥 **执行方分工**：区分肆瓦、MC两个执行方的职责
- 💡 **悬停交互详情**：鼠标悬停显示邮件模板、平台地址、参考文件等详细信息
- 📧 **邮件模板展示**：完整的报数邮件和打款凭证邮件模板
- 🔍 **缩放查看**：支持放大、缩小、重置视图

**浦东DRC项目流程：**
1. 每月1号或15号：肆瓦邮件报数 + MC运营平台上传Excel
2. 自动扣款规则：12点前上传当日6点扣款，12点后次日扣款

**西安/南京RBO项目流程：**
- 第一阶段（每月1号/15号）：肆瓦邮件报数 + MC提报OA
- 第二阶段（每月2号/16号）：肆瓦邮件报送打款凭证 + MC提报OA打款材料

**功能亮点：**
- 🎨 渐变色彩设计，执行方区分明显
- 📱 响应式布局，完美适配各种设备
- ⚡ 流畅交互动画，操作体验优秀
- 🔗 从主页一键跳转访问

## 📖 使用指南

### 1. RBO登记管理器使用

#### 新增记录
1. 点击"新增记录"按钮
2. 填写14个字段（带*为必填项）
3. 点击"保存"按钮

#### 搜索和筛选
- 在搜索框输入RBO编码或项目简称
- 使用状态下拉菜单筛选（已完成/处理中/待处理）
- 使用日期选择器筛选特定日期的记录

#### 编辑和删除
- 点击表格行操作列的"编辑"按钮修改记录
- 点击"删除"按钮删除记录（需确认）

#### Excel导入导出
- **导出**：点击"导出Excel"按钮，下载所有记录
- **导入**：点击"导入Excel"按钮，选择Excel文件批量导入

**详细使用说明**: 查看 [RBO_MANAGER_GUIDE.md](RBO_MANAGER_GUIDE.md)

---

### 2. RBO流程图使用

#### 项目切换
- 点击顶部标签页切换不同项目流程

#### 查看详情
- 将鼠标悬停在流程节点上
- 弹出窗口显示详细信息（邮件模板、平台地址等）

#### 视图控制
- 点击右上角"+"放大流程图
- 点击"-"缩小流程图
- 点击"重置"恢复默认视图

**详细使用说明**: 查看 [RBO_USAGE.md](RBO_USAGE.md)

## 💡 典型使用场景

### 场景1：月初报数登记
```
1. 查看流程图 → 了解每月1号的报数要求
2. 执行报数工作 → 按流程图中的邮件模板发送
3. 打开管理器 → 新增一条记录
4. 填写信息 → 处理日期、RBO编码、分成周期等
5. 保存记录 → 数据自动保存
```

### 场景2：月度数据统计
```
1. 打开管理器
2. 使用日期筛选 → 选择本月1号
3. 查看统计面板 → 本月处理的记录数和金额
4. 导出Excel → 生成月度报告
5. 在Excel中进行详细分析
```

### 场景3：批量数据导入
```
1. 准备Excel文件（包含所有14个字段）
2. 打开管理器 → 点击"导入Excel"
3. 选择文件 → 确认导入
4. 数据自动添加到系统中
```

## 📊 数据说明

### 数据存储
- 所有数据保存在浏览器的 **LocalStorage** 中
- 数据会自动保存，无需手动保存
- 即使关闭浏览器，数据也不会丢失
- 建议定期导出Excel文件作为备份

### 数据备份建议
- **每周导出**：定期导出Excel文件保存
- **重要操作前**：在批量删除或修改前先导出备份
- **云端备份**：将导出的Excel文件保存到云盘
