# 超级加盟商投资决策平台 v2.2

## 🎯 项目简介

这是一个**交互式投资决策平台**，专为超级加盟商行业设计。通过填入关键信息，实时计算投资回报、评估企业价值、生成标准化合同，帮助投资决策者快速完成投资分析。

## 📊 数据说明

### 投资回报计算公式
- **月均分成** = 月收入 × 分成比例
- **回本周期** = 投资金额 ÷ 月均分成
- **封顶金额** = 投资金额 × (1 + 年化收益率)
- **总回报** = 月均分成 × 联营期限
- **利润** = 总回报 - 投资金额

### 企业评分标准
- **总分**：200分
- **行业通识标准**：60分（30%权重）
  - 每项20分，共3项
- **专属分析标准**：140分（70%权重）
  - 每项30分，共5项

### 评级标准
- ⭐⭐⭐⭐⭐ **优秀**：170-200分，强烈推荐
- ⭐⭐⭐⭐ **良好**：140-169分，推荐投资
- ⭐⭐⭐ **一般**：110-139分，谨慎考虑
- ⭐⭐ **不推荐**：<110分，暂缓投资
