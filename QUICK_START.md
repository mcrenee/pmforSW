## 📊 下半部分：登记数据

### 1. 新增记录
```
点击"新增记录" → 填写14个字段 → 保存
```

**必填字段**：
- 处理日期
- 状态（已完成/处理中/待处理）
- 系统（如：二合一）
- RBO编码（如：RT-CN0100-PAWSV0001）
- 内部RBO编码（如：R0000000003）
- 简称（如：上海肆瓦南京机场）
- 分成周期开始/结束日
- 应打款金额
- 实际打款金额

### 2. 搜索筛选
```
搜索框：输入RBO编码或项目简称
状态筛选：全部/已完成/处理中/待处理
日期筛选：选择特定日期
```

### 3. 编辑/删除
```
点击表格中的"编辑"按钮 → 修改内容 → 保存
点击"删除"按钮 → 确认删除
```

### 4. 导入导出
```
导出：点击"导出Excel" → 下载文件
导入：点击"导入Excel" → 选择文件 → 确认导入
```

## 💡 常用场景

### 场景1：每月1号报数（浦东DRC）
1. **查看流程**（上半部分）
   - 点击"浦东DRC项目"标签
   - 查看肆瓦的邮件模板
   - 查看MC的平台地址

2. **执行任务**
   - 肆瓦：按邮件模板发送报数邮件
   - MC：登录平台上传Excel（注意12点规则）

3. **登记信息**（下半部分）
   - 点击"新增记录"
   - 填写：处理日期、RBO编码、分成周期、应打款金额等
   - 状态选择"处理中"
   - 保存

---

### 场景2：每月2号打款（西安/南京RBO）
1. **查看流程**（上半部分）
   - 点击"西安/南京RBO项目"标签
   - 查看第二阶段流程

2. **执行任务**

   - 点击"编辑"

## 🆘 常见问题

**Q: 数据会丢失吗？**
A: 数据保存在浏览器本地存储，关闭浏览器不会丢失。但清除浏览器缓存会导致数据丢失，建议定期导出Excel备份。

**Q: 可以同时打开多个标签页吗？**
A: 可以，但建议使用一个标签页操作，避免数据冲突。

**Q: 如何找回误删的记录？**
A: 误删无法恢复，建议在重要操作前先导出Excel备份。

**Q: Excel导入失败怎么办？**
A: 确保Excel文件包含所有14个字段，字段名称与系统一致。

**Q: 邮件模板中的【  】如何填写？**
A: 【  】表示需要填写的内容，如RBO编码、金额、日期等，根据实际情况填写。
