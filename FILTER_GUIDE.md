# 高级筛选功能使用指南

## 功能概述
数据登记系统现在支持任意字段的高级筛选功能，可以通过多个条件组合筛选数据记录。

## 基础筛选功能

### 1. 搜索框
- **位置**：数据表格上方
- **功能**：在所有字段中搜索关键词
- **用法**：输入关键词（如RBO编码、简称等），系统自动筛选

### 2. 状态筛选
- **选项**：全部状态、已完成、处理中、待处理
- **用法**：从下拉菜单选择状态

### 3. 系统筛选
- **选项**：全部系统、二合一、全流程
- **用法**：从下拉菜单选择系统类型

## 高级筛选功能

### 如何使用

1. **打开高级筛选面板**
   - 点击"高级筛选"按钮
   - 面板会展开，显示筛选条件区域

2. **添加筛选条件**
   - 点击"添加条件"按钮
   - 每个条件包含三部分：
     * **字段选择**：选择要筛选的字段
     * **运算符**：选择筛选方式
     * **筛选值**：输入筛选的值

3. **可筛选的字段**
   - 处理日期
   - 状态
   - 系统
   - RBO编码
   - 内部RBO编码
   - 简称
   - 交易账号
   - 周期开始日
   - 周期结束日
   - 信息流OA
   - 信息流日期
   - 应打款金额
   - 实际打款金额
   - 资金流OA单
   - 资金流日期

4. **支持的运算符**
   - **包含**：字段值包含输入的文字
   - **等于**：字段值完全等于输入的值
   - **开头是**：字段值以输入的文字开头
   - **结尾是**：字段值以输入的文字结尾
   - **大于**：字段值（数字）大于输入的值
   - **小于**：字段值（数字）小于输入的值

## 使用示例

### 示例1：查找特定RBO的已完成记录
1. 点击"高级筛选"
2. 添加条件1：RBO编码 - 包含 - RT-CN0100
3. 添加条件2：状态 - 等于 - 已完成
4. 自动显示筛选结果

### 示例2：查找大额打款记录
1. 点击"高级筛选"
2. 添加条件：实际打款金额 - 大于 - 100000
3. 自动显示所有大于10万的记录

### 示例3：查找特定时间段的记录
1. 点击"高级筛选"
2. 添加条件1：处理日期 - 大于 - 2025-08-01
3. 添加条件2：处理日期 - 小于 - 2025-09-01
4. 显示8月份的所有记录

### 示例4：查找西安机场的处理中记录
1. 点击"高级筛选"
2. 添加条件1：简称 - 包含 - 西安
3. 添加条件2：状态 - 等于 - 处理中

## 组合筛选

高级筛选可以与基础筛选组合使用：
- 基础筛选（搜索框、状态、系统）
- + 高级筛选（多个自定义条件）
- = 精确的筛选结果

## 清除筛选

点击"清除筛选"按钮可以一次性清除：
- 搜索框的内容
- 状态筛选
- 系统筛选
- 所有高级筛选条件

系统会恢复显示全部数据。

## 注意事项

1. **实时筛选**：每次修改条件后，结果会自动更新
2. **多条件组合**：所有条件都必须同时满足（AND逻辑）
3. **删除条件**：点击条件右侧的"×"按钮可删除单个条件
4. **数字比较**：使用"大于"/"小于"运算符时，确保输入的是数字

## 导出功能

筛选后的数据可以直接导出为Excel文件：
- 点击"导出Excel"按钮
- 只导出当前筛选结果
- 文件名格式：RBO数据_YYYY-MM-DD.xlsx
