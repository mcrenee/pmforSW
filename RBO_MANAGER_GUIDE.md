# RBO登记管理器使用说明

## 📖 系统概览

RBO登记管理器是一个**专业的报数打款登记与追踪系统**，用于记录和管理浦东DRC、西安RBO、南京RBO三个项目的所有RBO处理记录。

## 🚀 快速访问

### 从主平台访问
从投资决策平台主页导航栏点击 **"RBO管理器"** 进入

### 直接访问
访问地址：`rbo-manager.html`

## 💡 核心功能详解

### 1. 📊 统计面板

页面顶部显示四个关键统计指标：

- **总记录数**：所有RBO处理记录的总数
- **已完成**：状态为"已完成"的记录数
- **处理中**：状态为"处理中"的记录数
- **总打款金额**：所有记录的实际打款金额总和

统计数据会随着记录的增删改自动更新。

---

### 2. 🔍 搜索与筛选

**搜索功能：**
- 在搜索框输入关键词
- 支持搜索：RBO编码、内部RBO编码、项目简称
- 实时搜索，输入即显示结果

**筛选功能：**
- **状态筛选**：选择"已完成"、"处理中"或"待处理"
- **日期筛选**：选择特定处理日期
- 支持多条件组合筛选

---

### 3. ✍️ 新增记录

点击 **"新增记录"** 按钮，填写以下信息：

#### 必填字段（标注 * ）

| 字段 | 说明 | 示例 |
|------|------|------|
| 处理日期 * | 处理的日期 | 2025-08-06 |
| 状态 * | 选择：已完成/处理中/待处理 | 已完成 |
| 系统 * | 所属系统名称 | 二合一 |
| RBO编码 * | 外部RBO编码 | RT-CN0100-PAWSV0001 |
| 内部RBO编码 * | 内部管理编码 | R0000000003 |
| 简称 * | 项目简称 | 上海肆瓦南京机场（二合一） |
| 分成周期开始日 * | 分成周期起始日期 | 2025-07-30 |
| 分成周期结束日 * | 分成周期结束日期 | 2025-07-31 |
| 应打款金额 * | 应支付的金额 | 0.00 |
| 实际打款金额 * | 实际支付的金额 | 0.00 |

#### 可选字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 信息流OA | 信息流OA单号 | IT06-20250801001 |
| 信息流处理日期 | 信息流的处理日期 | 2025-08-06 |
| 资金流OA单 | 资金流OA单号 | / 表示无 |
| 资金流处理日期 | 资金流的处理日期 | 2025-08-06 |

填写完成后点击 **"保存"** 按钮。

---

### 4. ✏️ 编辑记录

在表格的"操作"列点击 **编辑按钮** <i class="fas fa-edit"></i>：
- 弹出编辑窗口，显示当前记录的所有信息
- 修改需要更改的字段
- 点击 **"保存"** 保存修改

---

### 5. 🗑️ 删除记录

在表格的"操作"列点击 **删除按钮** <i class="fas fa-trash"></i>：
- 系统会弹出确认对话框
- 确认后记录将被永久删除
- 删除后统计数据自动更新

---

### 6. 📥 导出Excel

点击右上角的 **"导出Excel"** 按钮：
- 系统会将所有记录导出为Excel文件
- 文件名格式：`RBO登记记录_2026-02-05.xlsx`
- 包含所有14个字段
- 金额字段自动格式化为货币格式

**导出的Excel包含以下列：**
- 处理日期
- 状态
- 系统
- RBO编码
- 内部RBO编码
- 简称
- 对应分成周期开始日
- 对应分成周期结束日
- 信息流OA
- 信息流处理日期
- 应打款金额
- 实际打款金额
- 资金流OA单
- 资金流处理日期

---

### 7. 📤 导入Excel

点击右上角的 **"导入Excel"** 按钮：

**导入步骤：**
1. 点击"导入Excel"按钮
2. 选择要导入的Excel文件（.xlsx 或 .xls）
3. 系统会显示导入记录数量的确认对话框
4. 确认后，数据会添加到现有记录的前面
5. 自动更新表格和统计数据

**Excel格式要求：**
- 必须包含表头行（中文列名）
- 列名必须与导出的Excel一致
- 日期格式：YYYY-MM-DD
- 金额可以包含￥和逗号，系统会自动处理

**示例Excel格式：**
```
处理日期 | 状态 | 系统 | RBO编码 | ...
2025-08-06 | 已完成 | 二合一 | RT-CN0100-PAWSV0001 | ...
```

## 📋 数据管理

### 数据存储
- 所有数据保存在浏览器的 **LocalStorage** 中
- 数据会自动保存，无需手动保存
- 即使关闭浏览器，数据也不会丢失
- 建议定期导出Excel文件作为备份

### 数据清空
如需清空所有数据：
1. 打开浏览器开发者工具（F12）
2. 进入 Console（控制台）
3. 输入：`localStorage.removeItem('rbo_records')`
4. 刷新页面

### 数据备份建议
- **每周导出**：定期导出Excel文件保存
- **重要操作前**：在批量删除或修改前先导出备份
- **云端备份**：将导出的Excel文件保存到云盘

## 🎨 界面说明

### 状态标识

系统使用彩色标签标识不同状态：

- 🟢 **已完成**：绿色标签，表示流程已全部完成
- 🔵 **处理中**：蓝色标签，表示正在处理中
- 🟡 **待处理**：黄色标签，表示尚未开始处理

### 表格操作

- **悬停高亮**：鼠标悬停在表格行上会显示浅灰色背景
- **横向滚动**：表格内容较多时支持横向滚动查看
- **固定表头**：滚动时表头固定在顶部

## 💡 使用技巧

### 快速查找记录
1. 使用搜索框输入RBO编码或项目简称
2. 结合状态筛选缩小范围
3. 使用日期筛选查看特定日期的记录

### 批量操作
1. 先导出Excel文件
2. 在Excel中批量编辑数据
3. 清空系统中的旧数据
4. 重新导入修改后的Excel

### 数据核对
1. 使用"已完成"状态筛选查看已完成的记录
2. 检查实际打款金额与应打款金额是否一致
3. 确认信息流和资金流的OA单号和处理日期

### 月度统计
1. 使用日期筛选选择月初或月中
2. 查看该时间段的记录数和打款金额
3. 导出Excel进行详细分析

## ⚠️ 注意事项

### 数据填写规范
1. **日期格式**：统一使用YYYY-MM-DD格式
2. **金额格式**：输入纯数字，保留两位小数
3. **编码规范**：RBO编码按统一格式填写
4. **状态选择**：根据实际进度准确选择状态

### 数据安全
1. 定期导出备份，防止数据丢失
2. 不要在多个浏览器中同时操作
3. 清除浏览器缓存前务必先导出数据

### 浏览器兼容性
- **推荐浏览器**：Chrome、Edge、Firefox最新版
- **不支持**：IE浏览器
- **移动端**：支持但建议使用电脑端操作
